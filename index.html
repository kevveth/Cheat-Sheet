Semantic HTML Reference

Table Tags

Tag Description
<header> A container usually for either navigational links or introductory content
<nav> Used to define a block of navigation links such as menus and tables of contents.
<main> Encapsulates the dominant content within a webpage.
<footer> The content at the bottom of the subject information.
<section> Defines elements in a document, such as chapters, headings, or any other area of the document with the same theme.
<article> Holds content that makes sense on its own such as articles, blogs, comments, magazines, etc.
<aside> Used to mark additional information that can enhance another element but isnâ€™t required in order to understand the main content.
<figure> Used to encapsulate media such as an image, illustration, diagram, code snippet, etc, which is referenced in the main flow of the document.
<figcaption> Used to describe the media in the <figure> tag.
<audio> Used to embed audio content into a document.
<video> Used to add videos to our website.
<embed> Can embed any media content including videos, audio files, and gifs from an external source. Note that is a deprecated tag and other alternatives, such as <video>, <audio> and <img>, should be used in its place.

Attributes

Tag Description
controls automatically displays the media controls into the browser.
